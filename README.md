# Voice Scientific Calculator

A modern responsive **voice-enabled scientific calculator** designed for both **desktop and mobile browsers**.  
The calculator supports **keyboard, mouse, touch input, and voice output**, making it flexible and convenient for various use cases.

It includes multiple themes, a scientific function panel, calculation history, keyboard shortcuts, and responsive UI.

---

# Features

### Core Features
- Standard arithmetic operations
- Scientific functions
- Responsive UI (Desktop + Mobile)
- Calculation history
- Voice output for results
- Multi-input support
- Dynamic layout scaling

### Input Methods
The calculator can be controlled using:

- Keyboard
- Mouse
- Touch screen
- Numpad

### Scientific Functions
Accessible via the **Scientific Panel**.

Includes:

- sin()
- cos()
- tan()
- sqrt()
- log()
- ln()
- π (pi)
- e
- power (^)

---

# User Interface Features

### Responsive Layout
The calculator automatically adjusts based on screen size.

Desktop:
- History panel appears beside calculator.

Mobile:
- History appears below calculator.

### Themes
Three themes are included:

- Dark Theme
- Light Theme
- Neon Theme

Themes can be changed through the **menu**.

---

# Controls

### On-Screen Controls

| Button | Function |
|------|------|
| = | Calculate result |
| C | Clear expression |
| ⌫ | Delete last input |
| ^ | Power operation |
| ☰ | Open menu |
| 🎤 | Voice input |

---

# Keyboard Shortcuts

The calculator supports several keyboard shortcuts.

| Key | Action |
|----|----|
| Enter | Calculate expression |
| Space | Insert space |
| Backspace | Delete last character |
| Esc | Clear calculator |
| H | Toggle history panel |
| T | Open theme selector |
| M | Open main menu |
| ^ | Power operation |
| Numbers | Insert number |
| + - * / | Arithmetic operators |
| ( ) | Parentheses |

---

# Menu Options

The menu provides quick access to features:

- Theme Switcher
- History Toggle
- Scientific Panel Toggle
- Voice Output Toggle

Menus automatically close when clicking outside.

---

# Voice Output

The calculator can **speak the result aloud**.

Steps:

1. Open Menu
2. Enable **Voice Output**
3. Perform a calculation

The calculator will say:

> "The result is ..."

This feature uses the browser **SpeechSynthesis API**.
